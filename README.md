# 📊 Financial Credit Risk Analysis Dashboard | Power BI

## 📌 Project Overview

This project is an interactive **Financial Credit Risk Analysis Dashboard** developed in **Microsoft Power BI**. The dashboard helps financial institutions analyze customer credit profiles, identify high-risk borrowers, monitor loan performance, and make data-driven lending decisions.

The dashboard provides a comprehensive overview of customer demographics, loan distribution, repayment behavior, and credit risk using dynamic visualizations and interactive filters.

---

## 🎯 Project Objectives

- Analyze customer credit risk.
- Identify high-risk and low-risk borrowers.
- Monitor loan portfolio performance.
- Track customer financial characteristics.
- Enable business users to make informed lending decisions.
- Improve financial decision-making through interactive dashboards.

## 📊 Business Insights

The dashboard helps answer important business questions such as:

- Which customer segments have the highest default risk?
- Which loan categories contribute the most to total lending?
- How does customer income impact credit risk?
- Which occupations have better repayment behavior?
- What is the distribution of customers across different risk levels?
- How do demographic factors influence loan performance?
